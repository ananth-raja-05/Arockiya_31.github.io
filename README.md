## âœï¸ Customization

### Update Content

1. **Personal Info**: Edit `index.html` - Update name, bio, and about sections
2. **Blog Posts**: Edit `blog.html` - Add/modify blog post cards
3. **Colors**: Edit `styles.css` - Change CSS variables at the top

### Color Scheme

Current theme uses soft blues and greys. To change:

Open `styles.css` and modify these variables:
```css
--primary: #5b9bd5;        /* Main blue */
--primary-dark: #4a7fb8;   /* Darker blue */
--primary-light: #7eb3e0;  /* Lighter blue */
--secondary: #94a3b8;      /* Grey accent */
```
